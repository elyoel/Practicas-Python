Primeros pasos:
Primero creamos un entorno virtual (Virtual Environment) para separar las instalaciones solo para nuestro proyecto
Ejecutaremos:
```
    python -m venv .venv
```
Luego deberemos activar y hacer uso del entono virtual ejecutaremos:
```
    .\.venv\Scripts\activate.bat
```

Cuando tendremos el entorno virtual creado y activado podremos instalar Jupyter Notebook
Haremos uso del comando:
```
    pip install jupyter[lab]
```
Una vez instalado podremos abrir jupyter notebook con el comando:
```
    jupyter lab
```
